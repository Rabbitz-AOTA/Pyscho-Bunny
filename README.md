Psycho Bunny Multi-Virus Toolkit

This repository contains a Python script that serves as a multi-virus toolkit. It allows users to select and launch various types of viruses, including boot sector viruses, web scripting viruses, browser hijackers, resident viruses, direct action viruses, polymorphic viruses, file infector viruses, and multipartite viruses.
Features

    Interactive menu system for selecting and launching viruses
    ASCII art banner displayed upon script execution
    Clear screen command used to enhance user experience
    Exit option in the main menu to terminate the script
    Modular structure for easy addition of new viruses

Usage

    Clone the repository to your local machine.
    Run the script using a Python interpreter.
    Select the desired virus from the main menu.
    The selected virus will be launched, and its functionality will be executed.
